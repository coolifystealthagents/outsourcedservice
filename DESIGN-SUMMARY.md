# Design summary

- Design concept: Service catalog
- Layout personality: minimal monochrome
- Typography: DM Serif Display headings with DM Sans body text
- Color palette: primary #111827, secondary #e5e7eb, accent #ef4444, background #ffffff, text #111827, muted #6b7280
- Shape language: Minimal use of containers
- Visual motif: monochrome catalog lines
- Unique layout choices: first sections use Service catalog, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
